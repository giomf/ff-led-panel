API for controlling AM03127 LED panels

Get a specific page

Retrieves the content and settings of a specific page by ID

path Parameters
pageId
required
string^[A-Z]$

Single uppercase letter (A-Z) identifying the page

Responses

Local development server

http://localhost:80/page/{pageId}

Response samples

  • 200
Content type
application/json
{
  • "line": 0,
  • "page": "string",
  • "leading": "block_move",
  • "lagging": "curtain_down",
  • "waiting_mode_and_speed": "fastest_blinking",
  • "message": "string"
}

Create or update a page

Creates a new page or updates an existing page with the specified ID

path Parameters
pageId
required
string^[A-Z]$

Single uppercase letter (A-Z) identifying the page

Request Body schema: application/json
required
text
required
string <= 16 characters

Text content to display on the page

leading
string (Leading)
Enum: "block_move" "curtain_down" "curtain_up" "hold" "immediate" "pen_amplus" "pen_hello_world" "pen_welcome" "random" "scroll_down" "scroll_left" "scroll_right" "scroll_up" "snow" "twinkle" "vclose" "vopen" "xopen"

Animation effect for page entry

lagging
string (Lagging)
Enum: "curtain_down" "curtain_up" "hold" "immediate" "scroll_down" "scroll_left" "scroll_right" "scroll_up" "vclose" "vopen" "xopen"

Animation effect for page exit

waiting_mode_and_speed
string (WaitingModeAndSpeed)
Enum: "fastest_blinking" "fastest_normal" "fastest_song1" "fastest_song2" "fastest_song3" "middle_fast_blinking" "middle_fast_normal" "middle_fast_song1" "middle_fast_song2" "middle_fast_song3" "middle_slow_blinking" "middle_slow_normal" "middle_slow_song1" "middle_slow_song2" "middle_slow_song3" "slowest_blinking" "slowest_normal" "slowest_song1" "slowest_song2" "slowest_song3"

Display mode and speed while waiting

month

Month

second
integer <uint8> [ 0 .. 59 ]

Second

year
integer <uint8>

Year (last two digits)

week
